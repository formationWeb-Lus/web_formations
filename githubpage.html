Configuration Semaine 1 : Outils — Hébergement — GitHub Pages

Présentation

GitHub Pages est une fonctionnalité qui permet d’héberger un dépôt GitHub comme un site web. Cela facilite le partage et la soumission de vos travaux déployés pour évaluation.

Voici les étapes pour activer GitHub Pages sur votre dépôt webacademy.

Instructions

  1. Connectez-vous à GitHub et ouvrez le dépôt webacademy.
  2. Cliquez sur Paramètres.
  3. Dans le menu de gauche, cliquez sur Pages.
  4. Dans Build and deployment :
    • Source : "Déployer depuis une branche"
    • Branche : main (ou master), chemin /(root), puis Enregistrer.

✅ GitHub Pages est activé : https://votre-utilisateur.github.io/webacademy

Vous devez avoir un fichier index.html pour que le site s’affiche. La publication peut prendre quelques minutes après activation.

Besoin d’aide ?

Si vous n’avez pas pu terminer ces étapes, demandez de l’aide sur votre plateforme de communication.
