Semaine 1 : Outils – Déploiement – Dépôt GitHub

Présentation

Cette activité vous permet de configurer votre dépôt wd300 sur GitHub et de le cloner sur votre ordinateur. Nous utiliserons GitHub pour stocker vos travaux et GitHub Pages pour héberger vos projets en ligne.

GitHub est une plateforme en ligne de gestion de versions et de collaboration. Elle offre les fonctionnalités de Git avec des outils supplémentaires pour le travail en équipe.

Instructions de l’activité

  1. Créez ou utilisez votre compte GitHub.
  2. Sur GitHub, créez un dépôt nommé wd300 (en minuscules).
  3. Vérifiez que Git est installé : 
    git --version
  4. Configurez votre nom d’utilisateur et votre e-mail global Git : 
    git config --global user.name "VotreNomUtilisateurGitHub"
    git config --global user.email "VotreEmail@example.com"
  5. Assurez-vous que Git ne tienne pas compte de la casse dans les noms de fichiers : 
    git config --global core.ignorecase false
  6. Clonez le dépôt sur votre ordinateur.
  7. Ouvrez le dossier du dépôt dans VS Code, modifiez des fichiers, validez (commit) et poussez (push) vers GitHub Pages.
