Introduction à la fonction print() en Python

1. Qu’est-ce que la fonction print() ?

La fonction print() permet d’afficher des informations dans la console. C’est la fonction de base pour communiquer avec l’utilisateur ou voir ce qui se passe dans un programme Python.

2. Syntaxe de base

print(valeur_ou_texte)

valeur_ou_texte peut être :

  • un texte (entre guillemets " " ou ' '),
  • un nombre,
  • une variable,
  • une expression.

3. Exemples simples

Exemple 1 : Afficher un message simple

# Affiche un message simple
print("Bonjour tout le monde !")

Exemple 2 : Afficher un nombre

# Affiche un nombre
print(42)

Exemple 3 : Afficher une addition

# Affiche le résultat de 2 + 3
print(2 + 3)

4. Afficher plusieurs éléments à la fois

On peut mettre plusieurs éléments séparés par des virgules. Python affiche chaque élément avec un espace entre eux.

print("Le résultat est", 5)
print("Bonjour", "les", "étudiants")

5. Afficher le contenu d’une variable

nom = "Alice"
print(nom)

6. Combiner texte et variables

Méthode 1 : avec des virgules

age = 30
print("J'ai", age, "ans")

Méthode 2 : avec une f-string (Python 3.6+)

nom = "Alice"
age = 30
print(f"Je m'appelle {nom} et j'ai {age} ans")

7. Quelques astuces avec print()

Changer la fin de ligne (supprimer le saut de ligne automatique) :

print("Bonjour", end=" ")
print("tout le monde !")
# Affiche : Bonjour tout le monde ! (sur une seule ligne)

Changer le séparateur entre plusieurs éléments :

print("2025", "08", "11", sep="-")
# Affiche : 2025-08-11

8. Exercices pratiques

  • Affiche ton prénom dans la console.
  • Affiche la somme de 12 et 5.
  • Crée une variable ville contenant ta ville, puis affiche J'habite à [ville].
  • Utilise une f-string pour afficher : Mon prénom est [prenom] et j'ai [age] ans.

9. Résumé

  • print() sert à afficher des informations.
  • On peut afficher du texte, des nombres, des variables ou des expressions.
  • Plusieurs éléments peuvent être affichés à la fois, séparés par des espaces ou un autre séparateur.
  • On peut contrôler la fin de ligne et le séparateur grâce aux arguments end et sep.
