Il est probable que vous ayez déjà Visual Studio Code installé. Cependant, vous devriez suivre cette activité pour vérifier les mises à jour et personnaliser votre installation.

Semaine 1 : Configuration des outils – Édition – Visual Studio Code

L’objectif de cette activité est d’installer VS Code. Il est gratuit et disponible sur Mac, Windows et Linux. C’est l’application utilisée pour écrire et modifier du code.

WebAcademy utilise VS Code comme éditeur principal car il est gratuit, open source, populaire et dispose d’une grande communauté de support et d’extensions.

Instructions

  1. Installez VS Code si ce n’est pas déjà fait : Visual Studio Code.
  2. Si vous l’avez déjà, vérifiez les mises à jour : Menu Aide ► Vérifier les mises à jour...
  3. Ajoutez l’extension Live Server pour tester localement dans votre navigateur.
  4. Personnalisez votre installation :
    • Activer formatage du code à l’enregistrement.
    • Activer enregistrement automatique après un délai.
    • Activer mise à jour automatique des balises HTML liées.
    • Désactiver la saisie semi-automatique IA en ligne.
    • Découvrir Emmet pour les abréviations HTML et CSS.
