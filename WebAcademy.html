Semaine 1 : Configuration des outils – Édition – Visual Studio Code et python

L’objectif de cette activité est d’installer VS Code et python. Il est gratuit et disponible sur Mac, Windows et Linux. C’est l’application utilisée pour écrire et modifier du code.

WebAcademy utilise VS Code comme éditeur principal car il est gratuit, open source, populaire et dispose d’une grande communauté de support et d’extensions.

Installation de Python 3 et Visual Studio Code

Cette semaine, vous installerez sur votre ordinateur les outils essentiels pour programmer tout au long du semestre :

  • Visual Studio Code (VS Code) : l’éditeur de code pour écrire vos programmes.
  • Python 3 : l’interpréteur pour exécuter vos programmes Python.

Les programmeurs passent beaucoup de temps à chercher des solutions en ligne. Ce cours vous aidera à :

  • Comprendre les bases de la programmation.
  • Apprendre et appliquer de nouvelles connaissances.
  • Utiliser efficacement les ressources disponibles sur Internet.

Conseils importants :

  • Téléchargez Python uniquement depuis python.org.
  • Évitez d’installer Python via Microsoft Store, Anaconda, Homebrew, ou directement depuis VS Code.
  • Après installation, vérifiez que Python fonctionne en ouvrant un terminal et en tapant python --version ou python3 --version.
  • Installez l’extension Python dans VS Code pour bénéficier d’outils comme l’autocomplétion et le débogage.

Prochaine étape

Une fois ces outils installés, vous serez prêt(e) à écrire et exécuter vos premiers programmes Python dès la semaine prochaine.

Instructions

  1. Installez VS Code si ce n’est pas déjà fait : Visual Studio Code.
  2. Si vous l’avez déjà, vérifiez les mises à jour : Menu Aide ► Vérifier les mises à jour...
  3. Ajoutez l’extension Live Server pour tester localement dans votre navigateur.
  4. Personnalisez votre installation :
    • Activer formatage du code à l’enregistrement.
    • Activer enregistrement automatique après un délai.
    • Activer mise à jour automatique des balises HTML liées.
    • Désactiver la saisie semi-automatique IA en ligne.
    • Découvrir Emmet pour les abréviations HTML et CSS.
